# Price Comparison (Сравнение цен)

## Описание

Навык автоматического сравнения цен на один и тот же товар на разных платформах. Позволяет находить лучшие предложения, учитывая скидки, купоны, стоимость доставки и итоговую цену.

## Связанные навыки

- `product-search.md` — Поиск товара для сравнения
- `data-extraction.md` — Извлечение цен и условий

---

## Инструменты

- **`search_web`** — Поиск товара на разных платформах
- **`tabs_create`** — Параллельное сравнение
- **`get_page_text`** — Извлечение цен и условий

---

## Алгоритм

### 1. Идентификация товара
- Определить модель, артикул, UPC/EAN

### 2. Поиск на площадках
- Открыть 5-10 площадок одновременно

### 3. Сбор цен
- Базовая цена
- Цена со скидкой
- Стоимость доставки

### 4. Сравнение
- Рассчитать итоговую цену
- Ранжировать по выгодности

---

## Пример

```
Товар: iPhone 15 Pro 256GB

Amazon: $1099 + $0 доставка = $1099
Best Buy: $1149 - $50 скидка + $0 = $1099
eBay: $1050 + $15 доставка = $1065 ✅

Лучшее предложение: eBay
```

## Лучшие практики

1. Проверять подлинность товара
2. Учитывать все дополнительные расходы
3. Проверять репутацию продавца
4. Сохранять скриншоты цен
5. Проверять историю цен
