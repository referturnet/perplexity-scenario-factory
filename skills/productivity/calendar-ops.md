# Calendar Operations (Операции с календарём)

## Описание

Навык автоматизации работы с календарями: создание, редактирование и поиск событий. Позволяет управлять встречами, отслеживать занятость и планировать расписание.

## Связанные навыки

- `email-management.md` — Парсинг приглашений из писем
- `form-filling.md` — Заполнение форм событий

---

## Инструменты

- **`navigate`** — Открытие Google Calendar / Outlook Calendar
- **`find`** — Поиск событий, кнопок
- **`form_input`** — Ввод названий, дат, участников
- **`computer`** — Клики по событиям, таймслотам

---

## Алгоритм

### 1. Поиск свободного времени
- Открыть календарь на нужную неделю
- Проанализировать занятые слоты

### 2. Создание события
- Клик на свободное время
- Заполнить название, время, место
- Добавить участников

### 3. Напоминания
- Установить уведомления
- Настроить рекуррентные события

---

## Пример

```
Задача: Запланировать встречу на 30 мая в 10:00

Действия:
1. Открыть Google Calendar
2. Перейти на 30 мая
3. Клик на 10:00
4. Заполнить: “Встреча с клиентом”
5. Добавить email участника
6. Сохранить
```

---

## Лучшие практики

1. Не создавать встречи без подтверждения
2. Проверять часовые пояса
3. Указывать место проведения
4. Добавлять описание и ссылки
5. Уважать личные календари других
