# Taxonomy — Classification System

**Последнее обновление:** 2026-05-29

> Система классификации для 200 сценариев использования Perplexity.

---

## 🏷 Оси классификации

### Ось 1: Категория (CATEGORY)

| Код | Категория | Описание |
|-----|-----------|----------|
| `RES` | Research & Analysis | Исследования, анализ, факт-чекинг |
| `SHP` | Shopping & E-commerce | Покупки, сравнение, поставщики |
| `ADM` | Admin & Operations | Операционные задачи, автоматизация |
| `DEV` | Development & Learning | Разработка, обучение, код |
| `MKT` | Marketing & Content | Маркетинг, контент, SEO |
| `FIN` | Finance & Analytics | Финансы, аналитика данных |
| `PRD` | Productivity & Misc | Продуктивность, разное |

### Ось 2: Сложность (COMPLEXITY)

| Уровень | Описание | Кол-во шагов |
|---------|----------|---------------|
| ⭐ | Простой | 1-2 запроса |
| ⭐⭐ | Средний | 3-5 запросов |
| ⭐⭐⭐ | Сложный | 6+ запросов, цепочки |

### Ось 3: Тип вывода (OUTPUT)

| Код | Тип | Описание |
|-----|-----|----------|
| `RPT` | Report | Структурированный отчёт |
| `TBL` | Table | Сравнительная таблица |
| `LST` | List | Список вариантов |
| `TXT` | Text | Текст, статья, описание |
| `ACT` | Action | Набор действий, план |
| `DAT` | Data | Числовые данные, метрики |

### Ось 4: Источник данных (SOURCE)

| Код | Источник |
|-----|----------|
| `WEB` | Веб-поиск (Perplexity default) |
| `API` | Perplexity Sonar API |
| `DOC` | Загруженный документ |
| `MEM` | Memory / контекст сессии |
