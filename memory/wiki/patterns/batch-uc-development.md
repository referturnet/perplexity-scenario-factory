# Pattern: Batch UC Development

**Категория:** patterns  
**Теги:** batch, uc, workflow, scale  
**Обновлено:** 2026-05-29

## Контекст

Когда каталог должен вырасти до 200 сценариев, одиночное создание файлов становится слишком медленным.

## Паттерн

Разработка идёт сериями по 5–10 UC:
1. выбрать тематический блок;
2. определить названия и цели;
3. создать guides в едином формате;
4. добавить одну wiki-страницу с выводом по серии;
5. обновить progress/changelog/master table.

## Когда использовать

- При масштабировании каталога.
- При работе несколькими сессиями.
- Когда важно поддерживать единый стандарт качества.
