# UC-005: Trend Analysis — Анализ тренда / hype cycle

**Категория:** Research & Analysis (RES)  
**Сложность:** ⭐⭐  
**Вывод:** RPT + TBL  
**Источник:** WEB  
**Обновлено:** 2026-05-29

## Цель

Понять, находится ли тема на стадии роста, перегрева, стабилизации или спада.

## Входные данные

- Тема или технология.
- Период анализа.
- Регион или глобальный фокус.

## Шаги выполнения

### Шаг 1: Сбор сигналов
Новости, упоминания, запуски, инвестиции, мнения экспертов.

### Шаг 2: Разделение сигналов
Рост интереса / реальные внедрения / маркетинговый шум / критика.

### Шаг 3: Оценка стадии
Early emergence, growth, hype, correction, practical adoption.

### Шаг 4: Практический вывод
Тренд временный или структурный, где реальные use cases.

## Выходные данные

- оценка стадии тренда;
- список сигналов за и против;
- practical interpretation для бизнеса.

## Пример промпта

```text
Проанализируй тренд: [тема].
Нужно:
1. Определить, растёт ли тренд или перегрет
2. Выделить признаки реального adoption
3. Отделить хайп от практической ценности
4. Указать риски и ограничения
5. Дать вывод: временный шум или устойчивое направление
```

## Критерии качества

- [ ] Есть признаки роста
- [ ] Есть признаки перегрева/рисков
- [ ] Есть оценка стадии тренда
- [ ] Есть practical вывод
