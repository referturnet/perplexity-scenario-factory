# Memory System — Index

**Последнее обновление:** 2026-05-29

> Система памяти проекта. Хранит накопленные знания по паттернам, сценариям и результатам исследований.
> Реализует паттерн LLM-Wiki по Karpathy: каждая страница — атомарный факт или паттерн, оптимизированный для поиска LLM.

---

## 📂 Структура

```
memory/
├── INDEX.md          ← этот файл
├── wiki/             ← атомарные статьи знаний
│   ├── README.md
│   ├── patterns/     ← паттерны промптов и запросов
│   ├── findings/     ← ключевые находки из исследований
│   └── concepts/     ← ключевые концепции
└── raw/              ← сырые материалы для обработки
    └── README.md
```

## 🔄 Правила обновления

1. После каждой сессии работы — добавить новые паттерны/находки
2. Каждый файл wiki — одна атомарная идея (max 200 строк)
3. Заголовки H2-H4 для LLM-навигации
4. Обязательные поля: **Контекст**, **Паттерн**, **Пример**, **Когда использовать**
5. Обновить эту таблицу при добавлении файлов
