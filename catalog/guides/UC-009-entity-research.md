# UC-009: Entity Research — Исследование персоны или компании

**Категория:** Research & Analysis (RES)  
**Сложность:** ⭐⭐  
**Вывод:** RPT  
**Источник:** WEB  
**Обновлено:** 2026-05-29

## Цель

Быстро собрать структурированный профиль компании, основателя, эксперта, бренда или организации.

## Входные данные

- Имя персоны или название компании.
- Контекст, зачем исследование нужно.
- География, если релевантно.

## Шаги выполнения

### Шаг 1: Собрать базовый профиль
Кто это, чем занимается, где работает.

### Шаг 2: Собрать репутационные сигналы
Упоминания в СМИ, интервью, продукты, проекты, отзывы, controversies.

### Шаг 3: Определить relevance
Почему эта компания/персона важна именно для заданного контекста.

### Шаг 4: Сформировать summary
Компактный профиль с ключевыми фактами, недосказанностями и open questions.

## Выходные данные

- профиль сущности;
- список ключевых фактов;
- репутационные сигналы;
- вопросы для дальнейшей проверки.

## Пример промпта

```text
Исследуй компанию / персону: [название].
Нужно:
1. Краткий профиль
2. Чем известны
3. Какие продукты, проекты или достижения есть
4. Какие есть репутационные сигналы или риски
5. Почему это важно в контексте [контекст]
```

## Критерии качества

- [ ] Есть базовый профиль
- [ ] Есть сигналы качества/риска
- [ ] Есть связь с контекстом
- [ ] Есть open questions
