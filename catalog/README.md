## Структура UC

Каждый сценарий содержит:
- **Цель** — что нужно получить
- **Входные данные** — что нужно подготовить
- **Шаги** — пошаговый алгоритм запросов
- **Выходные данные** — формат результата
- **Паттерны** — ссылки на переиспользуемые паттерны
