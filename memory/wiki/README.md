# Memory Wiki

> Атомарные статьи знаний для LLM-агентов. Каждая страница — один паттерн, концепция или находка.

## Принципы (Karpathy LLM-Wiki)

- **Атомарность:** одна идея = один файл
- **Поискуемость:** H2-H4 заголовки, ключевые слова в тексте
- **Практичность:** каждая статья содержит пример применения
- **Краткость:** max 200 строк на файл

## Структура статьи

```markdown
# [Название паттерна/концепции]

**Категория:** patterns | findings | concepts  
**Теги:** tag1, tag2  
**Обновлено:** YYYY-MM-DD

## Контекст
[Когда возникает, какую проблему решает]

## Паттерн / Суть
[Описание]

## Пример
[Конкретный пример применения]

## Когда использовать
[Условия применения]

## Связанные файлы
[Ссылки]
```

## Навигация

- [patterns/](./patterns/) — паттерны промптов
- [findings/](./findings/) — ключевые находки
- [concepts/](./concepts/) — базовые концепции
