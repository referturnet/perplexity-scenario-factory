# UC-010: Literature Review — Обзор литературы по теме

**Категория:** Research & Analysis (RES)  
**Сложность:** ⭐⭐⭐  
**Вывод:** RPT + TBL  
**Источник:** WEB  
**Обновлено:** 2026-05-29

## Цель

Сделать ускоренный обзор литературы: найти ключевые papers, авторов, дебаты, методы и направления дальнейшего чтения.

## Входные данные

- Тема исследования.
- Период публикаций.
- Уровень: intro / intermediate / advanced.
- Тип источников: academic, official, applied.

## Шаги выполнения

### Шаг 1: Map the terrain
Ключевые авторы, foundational papers, школы мысли, дебаты.

### Шаг 2: Group the literature
По темам, подходам, методам или аргументам.

### Шаг 3: Identify the best starting set
5–10 самых полезных материалов для старта.

### Шаг 4: Verify and refine
Проверить каждый источник вручную.

## Выходные данные

- карта литературы по теме;
- тематические кластеры;
- стартовый reading list;
- понимание основных дебатов и пробелов.

## Пример промпта

```text
Сделай обзор литературы по теме: [тема].
Нужно:
1. Найти foundational и recent papers
2. Выделить ключевых авторов и основные дебаты
3. Сгруппировать литературу по темам или подходам
4. Дать reading list для старта
5. Указать, что обязательно нужно проверять вручную
```

## Критерии качества

- [ ] Есть ключевые авторы и papers
- [ ] Есть тематические кластеры
- [ ] Есть reading list
- [ ] Есть указание на ручную проверку
- [ ] Есть gaps/disagreements
