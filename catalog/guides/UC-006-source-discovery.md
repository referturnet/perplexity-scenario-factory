# UC-006: Source Discovery — Поиск научных и первичных источников

**Категория:** Research & Analysis (RES)  
**Сложность:** ⭐⭐  
**Вывод:** LST + RPT  
**Источник:** WEB  
**Обновлено:** 2026-05-29

## Цель

Быстро находить первичные, научные, официальные или максимально надёжные источники.

## Входные данные

- Исследуемая тема.
- Тип нужных источников: papers, official docs, статистика, regulations.
- Период публикации.

## Шаги выполнения

### Шаг 1: Уточнить тип источника
Научные статьи, официальные документы, white papers, government data.

### Шаг 2: Запросить список источников
Самые важные первичные источники, последние исследования, авторы.

### Шаг 3: Отделить первичные источники от вторичных
Явно разделить source types.

### Шаг 4: Проверить пригодность
Краткая аннотация по каждому источнику.

## Выходные данные

- список надёжных источников;
- краткая аннотация;
- фильтрация по типу и качеству.

## Пример промпта

```text
Найди по теме [тема] лучшие первичные и научные источники.
Нужно:
1. Отдельно выделить official / academic / industry sources
2. Для каждого дать 1–2 предложения, о чём он
3. Указать, это первичный источник или пересказ
4. Приоритизировать самые полезные для старта исследования
```

## Критерии качества

- [ ] Источники разделены по типам
- [ ] Есть первичные источники
- [ ] Есть краткая аннотация
- [ ] Есть приоритизация
