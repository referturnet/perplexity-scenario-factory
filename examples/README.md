## Назначение

Golden Examples — это примеры, которые:
- Работают "out of the box"
- Показывают best practices
- Демонстрируют комбинацию skills + patterns
- Содержат полный код с комментариями

Примеры можно:
- Запускать непосредственно
- Адаптировать под свои задачи
