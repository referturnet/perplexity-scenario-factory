# UC-004: Market Sizing — Исследование рынка

**Категория:** Research & Analysis (RES)  
**Сложность:** ⭐⭐⭐  
**Вывод:** RPT + DAT  
**Источник:** WEB  
**Обновлено:** 2026-05-29

## Цель

Оценить размер рынка, динамику, ключевых игроков и потенциальную ёмкость ниши.

## Входные данные

- Ниша или продукт.
- География.
- Целевой сегмент.
- Горизонт оценки.

## Шаги выполнения

### Шаг 1: Определить рамки рынка
Продуктовая категория, аудитория, регион, B2B/B2C.

### Шаг 2: Собрать данные по объёму и росту
Market size, CAGR, revenue/GMV/user base.

### Шаг 3: Сопоставить несколько оценок
Показать диапазон, если источники не совпадают.

### Шаг 4: Разложить на сегменты
По гео, цене, типу клиента, use case.

### Шаг 5: Выделить возможности
Незанятые сегменты, риски, точки входа.

## Выходные данные

- оценка размера рынка;
- разбивка по сегментам;
- прогноз роста;
- список ключевых драйверов и ограничений.

## Пример промпта

```text
Оцени рынок: [ниша/продукт].
География: [регион].
Нужно:
1. Текущий размер рынка
2. Прогноз роста на 1–3 года
3. Ключевые сегменты
4. Основные игроки
5. Драйверы роста и риски
6. Где может быть возможность входа
```

## Критерии качества

- [ ] Есть оценка размера рынка
- [ ] Есть диапазон или объяснение расхождений
- [ ] Есть сегментация
- [ ] Есть драйверы/риски
- [ ] Есть вывод по возможности входа
