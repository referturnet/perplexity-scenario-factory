# Agent Skills

*Пока не создано*

Каталог с набором навыков (skills) для агентов браузерной автоматизации.

## Структура

```
skills/
├── README.md           # Этот файл
├── web/                # Веб-навигация и взаимодействие
│   ├── form-filling.md
│   ├── navigation.md
│   └── data-extraction.md
├── research/           # Исследовательские навыки
│   ├── deep-research.md
│   ├── competitive-analysis.md
│   └── fact-checking.md
├── commerce/           # E-commerce операции
│   ├── product-search.md
│   ├── price-comparison.md
│   └── order-tracking.md
└── productivity/       # Продуктивность
    ├── email-management.md
    ├── calendar-ops.md
    └── document-processing.md
```

## Назначение

Каждый файл skill описывает:
- Конкретную задачу или способность
- Пошаговый алгоритм выполнения
- Примеры использования
- Edge cases и обработку ошибок

## Как использовать

Skills можно комбинировать для создания сложных сценариев автоматизации.
