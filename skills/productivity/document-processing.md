# Document Processing (Обработка документов)

## Описание

Навык автоматизации работы с документами: чтение, редактирование, извлечение информации из Google Docs, Word Online, PDF. Позволяет обрабатывать большие объёмы текстовых данных.

## Связанные навыки

- `data-extraction.md` — Извлечение структурированных данных
- `navigation.md` — Навигация по документам

---

## Инструменты

- **`navigate`** — Открытие Google Docs, OneDrive
- **`get_page_text`** — Чтение содержимого документа
- **`find`** — Поиск текста, таблиц
- **`computer`** — Скролл, клики, копирование

---

## Алгоритм

### 1. Открытие документа
- Перейти по ссылке
- Дождаться загрузки

### 2. Чтение
- Извлечь весь текст
- Прокрутить для загрузки всего контента

### 3. Анализ
- Найти ключевую информацию
- Извлечь таблицы, списки

### 4. Структурирование
- Организовать данные
- Создать сводную таблицу

---

## Пример

```
Задача: Извлечь все email из Google Doc

Действия:
1. Открыть документ
2. get_page_text с query="email addresses"
3. Найти все шаблоны email
4. Создать список

Результат: [email1@test.com, email2@test.com, ...]
```

---

## Лучшие практики

1. Не редактировать без разрешения
2. Сохранять резервные копии
3. Уважать конфиденциальность
4. Проверять разрешения доступа
5. Логировать все операции
