## Что такое Pattern

Pattern — это проверенное решение типовой задачи, которое:
- Можно применять в разных контекстах
- Имеет четкую структуру реализации
- Описывает best practices
- Включает примеры кода

## Использование

Паттерны помогают строить надежные скрипты автоматизации без изобретения велосипеда.
