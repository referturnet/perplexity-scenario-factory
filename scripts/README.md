# Automation Scripts

*Пока не создано*

Каталог готовых скриптов автоматизации для Perplexity Browser Agent.

## Структура

```
scripts/
├── README.md              # Этот файл
├── web-scraping/         # Скрапинг данных
│   ├── extract-table.js
│   ├── scrape-listings.js
│   └── monitor-prices.js
├── testing/               # Автотестирование
│   ├── smoke-tests.js
│   ├── regression-suite.js
│   └── visual-tests.js
├── data-entry/           # Ввод данных
│   ├── bulk-upload.js
│   ├── form-automation.js
│   └── csv-importer.js
└── monitoring/           # Мониторинг
    ├── uptime-check.js
    ├── change-detector.js
    └── alert-manager.js
```

## Формат скриптов

Каждый скрипт содержит:
- Описание задачи
- Параметры запуска
- Код выполнения
- Примеры использования

## Использование

Скрипты можно использовать как готовые решения или как шаблоны для создания своих автоматизаций.
