# UC-001: Deep Research — Глубокое исследование темы

**Категория:** Research & Analysis (RES)  
**Сложность:** ⭐⭐⭐  
**Вывод:** RPT (Structured Report)  
**Источник:** WEB + API  
**Обновлено:** 2026-05-29

---

## 🎯 Цель

Получить всестороннее исследование по сложной теме: факты, тренды, конкуренты, мнения экспертов — в виде структурированного отчёта.

---

## 📥 Входные данные

- Тема исследования (1-3 предложения)
- Глубина (поверхностная / средняя / максимальная)
- Фокус: факты | тренды | мнения | сравнение
- Язык вывода

---

## 🔄 Шаги выполнения

### Шаг 1: Декомпозиция запроса

Разбить тему на 4-6 подзапросов:
```
Основная тема → [факты] [история] [тренды] [игроки] [мнения] [прогнозы]
```

### Шаг 2: Параллельные запросы

```
🔍 Запрос 1: "[тема] что это такое определение"
🔍 Запрос 2: "[тема] история развития 2020-2026"
🔍 Запрос 3: "[тема] ключевые игроки компании"
🔍 Запрос 4: "[тема] тренды 2025 2026"
🔍 Запрос 5: "[тема] критика проблемы ограничения"
🔍 Запрос 6: "[тема] прогнозы эксперты"
```

### Шаг 3: Синтез

Объединить результаты в структуру:
```markdown
# Отчёт: [Тема]

## Обзор
## Ключевые факты
## Хронология
## Игроки рынка
## Тренды
## Критика и ограничения
## Прогнозы
## Источники
```

---

## 📤 Выходные данные

Структурированный Markdown-отчёт с:
- Краткое резюме (3-5 предложений)
- 5+ ключевых фактов
- Таблица игроков/сравнений
- Список источников

- [ ] Структура H2-H4
