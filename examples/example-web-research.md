example-web-# Example: Web Research Automation

## Описание

Пример автоматизации веб-исследования с помощью Perplexity Browser Agent.

## Задача

Найти и собрать информацию о популярных фреймворках для браузерной автоматизации в 2026 году.

## Шаги выполнения

### 1. Поиск информации
```
Используй search_web для поиска по запросам:
- "browser automation frameworks 2026"
- "best web scraping tools"
- "headless browser comparison"
```

### 2. Анализ результатов
```
Пройди по топ-5 результатам:
- Открой каждую ссылку
- Извлеки ключевую информацию
- Сохрани в структурированном виде
```

### 3. Сравнительный анализ
```
Создай таблицу сравнения:
| Framework | Stars | Language | Features |
|-----------|-------|----------|----------|
| ...       | ...   | ...      | ...      |
```

## Ожидаемый результат

Структурированный отчет с:
- Списком фреймворков
- Сравнительной таблицей
- Рекомендациями
