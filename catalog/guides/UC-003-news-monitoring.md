# UC-003: News Monitoring — Мониторинг новостей по теме

**Категория:** Research & Analysis (RES)  
**Сложность:** ⭐⭐  
**Вывод:** RPT + LST  
**Источник:** WEB  
**Обновлено:** 2026-05-29

## Цель

Отслеживать новости и обновления по теме, компании, рынку или событию.

## Входные данные

- Тема или объект мониторинга.
- Период: день, неделя, месяц.
- Гео: глобально или конкретный регион.
- Формат: краткая сводка, список новостей, таблица изменений.

## Шаги выполнения

### Шаг 1: Сформулировать объект мониторинга
Определи сущность максимально конкретно.

### Шаг 2: Запросить актуальные обновления
Примеры: `последние новости [тема]`, `updates [company] latest news`.

### Шаг 3: Попросить структурировать результат
Требовать: 5–10 ключевых новостей, краткое описание, почему важно, ссылки, общий вывод.

### Шаг 4: Выделить сигналы
Follow-up: какие новости наиболее значимы, что повторяется, последствия.

## Выходные данные

- краткая новостная сводка;
- список ключевых событий;
- выводы по динамике темы.

## Пример промпта

```text
Сделай мониторинг новостей по теме: [тема].
Период: последние 7 дней.
Нужно:
1. Найти 7–10 самых важных новостей
2. Для каждой дать краткое описание
3. Объяснить, почему новость важна
4. В конце выделить 3 главных сигнала и общий тренд
```

## Критерии качества

- [ ] Есть период мониторинга
- [ ] Есть 5+ релевантных обновлений
- [ ] Есть вывод по значимости
- [ ] Есть ссылки на источники
- [ ] Есть общий тренд
