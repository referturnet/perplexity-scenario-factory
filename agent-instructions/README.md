# IDE Agent Files

*Пока не создано*

Этот каталог предназначен для хранения специализированных инструкций для IDE-агентов (Claude Code, Codex, Cursor, Windsurf).

## Структура

```
agent-instructions/
├── README.md              # Этот файл
├── claude-code.md         # Инструкции для Claude Code
├── cursor.md              # Инструкции для Cursor
├── windsurf.md            # Инструкции для Windsurf
└── shared/                # Общие инструкции для всех агентов
    ├── coding-standards.md
    ├── project-context.md
    └── workflow.md
```

## Назначение

Файлы в этом каталоге содержат:
- Контекст проекта для IDE-агентов
- Стандарты кодирования и архитектурные решения
- Специфичные для каждого агента инструкции и настройки
- Рабочие процессы и best practices

## Использование

1. Скопируйте соответствующий файл в корень вашего проекта
2. Переименуйте согласно требованиям вашего IDE
3. Адаптируйте под специфику вашего проекта
