# Email Management (Управление Email)

## Описание

Навык автоматизации работы с электронной почтой: чтение, фильтрация, поиск, сортировка и извлечение информации из писем. Позволяет обрабатывать большие объёмы корреспонденции и выполнять массовые операции.

## Связанные навыки

- `data-extraction.md` — Извлечение данных из писем
- `order-tracking.md` — Поиск трек-номеров

---

## Инструменты

- **`navigate`** — Открытие веб-версии почты
- **`get_page_text`** — Чтение содержимого писем
- **`find`** — Поиск кнопок, писем, фильтров
- **`computer`** — Клики по письмам, фильтрам, папкам

---

## Алгоритм

### 1. Подключение
- Открыть Gmail/Outlook web
- Проверить авторизацию

### 2. Поиск и фильтрация
- Применить фильтры (отправитель, дата, тема)
- Использовать поисковую строку

### 3. Чтение писем
- Открыть письмо
- Извлечь текст, вложения, метаданные

### 4. Извлечение данных
- Найти ключевую информацию
- Структурировать результаты

---

## Пример

```
Задача: Найти все письма от Amazon за последние 30 дней

Действия:
1. Поиск: "from:amazon.com after:2026/04/28"
2. Сортировка по дате
3. Извлечение номеров заказов
4. Формирование списка
```
